Starting with HTML

Published on April 12, 2023 • 5 min read

Today I began learning HTML. It's fascinating to see how websites are structured with various tags and elements. I started with the basic structure of an HTML document and learned about headings, paragraphs, links, and images.

What I found most interesting is how semantic HTML helps with accessibility and SEO. Using elements like <header>, <nav>, <main>, and <footer> makes the page structure clearer for both browsers and developers.

CSS Makes Everything Beautiful

Published on April 15, 2023 • 7 min read

I started learning CSS today. It's amazing how just a few lines of code can completely transform a webpage's appearance. I experimented with colors, fonts, spacing, and basic layout techniques.

The box model was initially confusing, but after practicing with padding, borders, and margins, it started to make sense. I also learned about different units of measurement (px, em, rem, %) and when to use each.

My First Responsive Design

Published on April 18, 2023 • 8 min read

Today I created my first responsive website that works well on both desktop and mobile devices. Media queries are powerful! I learned how to adjust layouts, font sizes, and even hide certain elements based on screen size.

I also discovered the importance of mobile-first design. Starting with the mobile layout and then adding media queries for larger screens seems to be the most efficient approach.
